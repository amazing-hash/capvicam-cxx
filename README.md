### capvicam-cxx
Video capture from usb camera via v4l2 subsystem
The project was implemented in order to compare the writing of the same project in C++ and Rust

#### Build:
```bash
git clone https://github.com/amazing-hash/capvicam-cxx.git
cd capvicam-cxx
mkdir build
cd build
cmake ..
make -j4
```
